Синхронізація цін і наявності

На цьому кроці роздрібний сайт читає товари прямо з Supabase. Ціни та наявність змінюй у Table Editor Supabase. Адмінку для редагування через базу підключу на наступному етапі.

Зараз використовується URL із конфігу проекту. Тут можна тимчасово підставити інший Apps Script URL лише для цього браузера.

# КОМПУТЕРРА — роздрібний сайт

Окремий проект для роздрібних клієнтів із каталогом, сторінкою товару, адмінкою та прямою синхронізацією цін через Google Sheets.

## Що вже є
- окремий публічний сайт без входу
- окрема адмінка `/admin/`
- адаптація під смартфони та ПК
- реальні фото та PDF-інструкції у проекті
- кнопка PDF автоматично неактивна, якщо файлу немає
- пряма синхронізація цін, наявності та ETA з Google Sheets
- ручна кнопка `Синхронізувати зараз` в адмінці
- нові товари з таблиці створюються автоматично
- товари, яких немає в таблиці, приховуються на публічному сайті

## Важливо
Ця версія **не використовує GitHub Actions** для цін.
Сайт читає опубліковану Google Таблицю **напряму** при відкритті сторінки.

## Структура проекту
- `index.html` — каталог
- `product.html` — сторінка товару
- `admin/index.html` — адмінка
- `assets/sheet-sync-config.js` — конфігурація Google Sheets
- `assets/sheet-sync.js` — логіка синхронізації з CSV
- `data/products.json` — базовий каталог сайту
- `data/retail-sheet-demo.csv` — demo CSV для перевірки логіки

## Тестовий вхід в адмінку

## Логіка синхронізації
Google Sheets керує:
- ціною
- наявністю
- датою прибуття
- автоматичним створенням нових товарів

Адмінка керує:
- описом
- фото
- PDF
- локальним приховуванням товару

Правила:
- новий товар із таблиці створюється автоматично;
- якщо товару немає в таблиці — він приховується на публічному сайті;
- якщо товар прихований в адмінці — він лишається прихованим у цьому браузері.

## Як працює оновлення цін
- ти змінюєш ціну в Google Sheets
- Google оновлює опублікований CSV
- сайт підтягує нові дані при відкритті або оновленні сторінки
- в адмінці можна натиснути `Синхронізувати зараз`, щоб перевірити відразу

## Публікація на GitHub Pages
1. Створи новий порожній репозиторій на GitHub.
2. Завантаж у нього **всі файли цього архіву**.
3. У `Settings` → `Pages` увімкни GitHub Pages для гілки `main`.
4. Відкрий сайт за виданим GitHub Pages URL.
5. Перевір зміну ціни, оновивши сторінку сайту.

## Важливо про локальне відкриття з архіву
Якщо відкривати `index.html` подвійним кліком із диска (`file://`), браузер може блокувати запит до Google Sheets.

Для повної перевірки синхронізації використовуй:
- GitHub Pages
- або локальний вебсервер, наприклад:

```bash
python -m http.server 8000
```

Після цього відкрий у браузері:

```text
http://localhost:8000
```

# Як увімкнути живу синхронізацію цін через Google Apps Script

Ця версія сайту **не використовує GitHub Actions і не читає Google CSV напряму**.
Надійна схема тут одна:

**Google Таблиця -> Google Apps Script Web App -> сайт на GitHub Pages**

## 1. Відкрий свою Google Таблицю
Відкрий саме ту таблицю, де є лист `Товари`.

## 2. Створи Apps Script
- У Google Таблиці натисни **Розширення -> Apps Script**
- Видали все, що там є
- Встав код із файлу `apps-script/Code.gs`
- Натисни **Save**

## 3. Розгорни скрипт як Web App
- Натисни **Deploy -> New deployment**
- Тип: **Web app**
- **Execute as:** Me
- **Who has access:** Anyone
- Натисни **Deploy**

Офіційно Apps Script web app публікується саме через `doGet`/`doPost`, а JSON можна віддавати через `ContentService`.

## 4. Скопіюй URL Web App
Після Deploy Google покаже URL. Скопіюй його.

## 5. Встав URL у проект
Є 2 способи.

### Варіант A — прямо у файлі проекту
Відкрий `assets/sheet-sync-config.js` і встав URL у `appsScriptUrl`.

### Варіант B — через адмінку
Після публікації сайту зайди в адмінку -> блок синхронізації -> встав URL -> натисни **Зберегти URL** -> **Синхронізувати зараз**.

## 6. Завантаж проект на GitHub Pages
Завантаж **усі файли з архіву** у новий репозиторій і увімкни Pages:
- Source: **Deploy from a branch**
- Branch: **main**
- Folder: **/(root)**

## 7. Як перевірити
- зміни ціну в Google Таблиці
- зачекай 1–3 хвилини
- відкрий сайт у режимі інкогніто або онови сторінку
- ціна має підтягнутися з таблиці

## Важливо
Без URL Apps Script жива синхронізація **не запрацює**. Сам по собі архів цього зробити не може, бо GitHub Pages — це статичний хостинг без бекенда.
